## 🚀 安装
- 请卸载旧版本后安装最新版本~
- 双击安装包后点击“开始安装”
- 接着安装进度条就会像喝了一杯昏睡红茶一样慢慢到达终点~

## ⏫ 放在U盘
- 下载绿色版后解压到U盘
